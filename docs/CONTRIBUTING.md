# Contributing

Thank you for your interest in contributing to ShareDrop.

Until **v1.0.0**, ShareDrop is maintained by the project maintainer(s). During this stage, the project's architecture and direction are still evolving, so code contributions are not being accepted.

Issues, bug reports, suggestions, and discussions are always welcome.

Once ShareDrop reaches **v1.0.0**, contribution guidelines will be updated to support community development.
